# Changelog

Все значимые изменения проекта фиксируются здесь. Формат ориентирован на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [Unreleased]

### Добавлено

- **Админ-панель:** отображается **только в браузере**; при открытии в Telegram (Mini App) скрыта (проверка `window.Telegram?.WebApp?.initData`). В браузере панель слева, контейнер игры с отступом 160px.
- **.gitignore:** расширен (node_modules, dist, .env, логи, ОС, IDE); CONCEPT.md исключён из репозитория; остальные внутренние документы можно раскомментировать при необходимости.
- **DEPLOY.md:** пошаговая инструкция — GitHub Desktop (первый коммит, пуш), деплой на GitHub Pages, создание бота в BotFather, описание/About/команды, подключение Mini App, кнопка «Играть».
- **BOT_CONTENT.md:** готовые тексты для бота — описание, About, команды, стартовое сообщение (драматично, иронично, кликбейт на основе «Об игре»); пример разметки кнопки web_app для Bot API.
- **AboutScene:** фон **about_bg.png** из `assets/sprites/environment/`; кремовая подложка под текст, иконка heart_broken в правом верхнем углу; кнопка «Вернуться в меню» на подложке; тёмный текст, межстрочный интервал 8.
- **Меню (MenuScene):** центральный блок на полупрозрачной подложке (15% alpha, тонкая обводка) с кнопками «Начать игру» (с левитацией) и «Закрыть»; подвал — «Об игре», «Поделиться»; под центральной панелью в одну строку — «Музыка вкл/выкл», «Эффекты вкл/выкл». Заголовок только на фоне (menu_bg), без отдельного текста. Единая палитра (#2c1810, #e8b4a8 и др.), тени у текста кнопок (offset 1, alpha 0.175).

### Изменено

- **BootScene:** загружается **about_bg** из `assets/sprites/environment/`; **люстра (chandelier) в коде не загружается** — она нарисована на **bg_room**.
- **GameScene:** сердца отображаются **поверх** Тараса (`setDepth(6)`); статус «Поймала!»/«Промах!» — выше по экрану (height×0.38), цвет **#2c1810**, жирный шрифт. Кнопка паузы — в стиле фона (скруглённая подложка, тёмный «II»). Пауза: оверлей с центральной панелью; **обратный отсчёт 3-2-1 по 0,5 с** на цифру (итого 1,5 с); панель скрыта во время отсчёта; текст отсчёта depth 200, цвет #e8e0d8.
- **WinScene / GameOverScene:** единый стиль UI (подложки, тёмный текст); кнопки плавно появляются (fade-in). В WinScene step4: Тарас и Катя **скрываются** при показе heart_float и taras_katya_hands (масштабы 1.25 и 1.1); дублирования персонажей нет.
- **Переход победа/проигрыш:** перед `scene.start('WinScene'|'GameOverScene')` вызываются `tarasSprite.destroy()`, `katyaSprite.destroy()`, `scene.stop('GameScene')` — дублирования спрайтов нет.

### Документация

- **STATUS.md:** актуализирован (about_bg, без chandelier в коде, UI меню/паузы/победы/проигрыша, глубины, отсчёт 0,5 с).
- **CONCEPT, GDD, SPEC, ARCHITECTURE, DEV_STEPS, ART_GUIDE, ASSETS.md, README:** приведены в соответствие с текущей реализацией (люстра на bg_room, about_bg, пауза 0,5 с, стиль UI).

### Код

- **src/objects/:** устаревшие TODO заменены на комментарии «реализовано в GameScene»; в ARCHITECTURE добавлено примечание, что объекты — legacy, логика в сценах.

### Планируется

- Деплой на GitHub Pages, Telegram Mini App (Шаг 8). Админ-панель в Telegram не показывается (оставлена только для браузера).
